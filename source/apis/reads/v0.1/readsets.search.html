Readsets: search

Gets a list of readsets matching the criteria.

Request

HTTP request

POST /readsets/search

Request body

In the request body, supply data with the following structure:

{
  "datasetIds": [
    string
  ],
  "name": string,
  "pageToken": string
}
Property name Value Description
datasetIds[] list Restricts this query to readsets within the given datasets. At least one ID must be provided.
pageToken string The continuation token, which is used to page through large result sets. To get the next page of results, set this parameter to the value of "nextPageToken" from the previous response.
name string Only return readsets with names matching this substring.

Response

If successful, this method returns a response body with the following structure:

{
  "readsets": [
    readsets Resource
  ],
  "nextPageToken": string
}
Property name Value Description
readsets[] list The list of matching Readsets.
nextPageToken string The continuation token, which is used to page through large result sets. Provide this value in a subsequent request to return the next page of results. This field will be empty if there aren't any additional results.
