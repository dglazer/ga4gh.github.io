Readsets: get

Gets a readset by ID.

 

Request

HTTP request

GET /readsets/readsetId

Parameters

Parameter name Value Description
Path parameters
readsetId string The ID of the readset.

Request body

Do not supply a request body with this method.

Response

If successful, this method returns a Readsets resource in the response body.
