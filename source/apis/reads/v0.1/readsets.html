Readsets

A Readset is a collection of Reads.

For a list of methods for this resource, see the end of this page.

Resource representations

{
  "id": string,
  "name": string,
  "datasetId": string,
  "created": long,
  "readCount": unsigned long,
  "fileData": [
    {
      "fileUri": string,
      "headers": [
        {
          "version": string,
          "sortingOrder": string
        }
      ],
      "refSequences": [
        {
          "name": string,
          "length": integer,
          "assemblyId": string,
          "md5Checksum": string,
          "species": string,
          "uri": string
        }
      ],
      "readGroups": [
        {
          "id": string,
          "sequencingCenterName": string,
          "description": string,
          "date": string,
          "flowOrder": string,
          "keySequence": string,
          "library": string,
          "processingProgram": string,
          "predictedInsertSize": integer,
          "sequencingTechnology": string,
          "platformUnit": string,
          "sample": string
        }
      ],
      "programs": [
        {
          "id": string,
          "name": string,
          "commandLine": string,
          "prevProgramId": string,
          "version": string
        }
      ],
      "comments": [
        string
      ]
    }
  ]
}
Property name Value Description Notes
created long The date this readset was created in milliseconds from the epoch.
datasetId string The ID of the dataset this readset belongs to.
fileData[] list File information from the original BAM import. See the BAM format specification for additional information on each field.
fileData[].comments[] list (@CO) One-line text comments.
fileData[].fileUri string The file uri that this data was imported from.
fileData[].headers[] list (@HD) The header line.
fileData[].headers[].sortingOrder string (SO) Sorting order of alignments.
fileData[].headers[].version string (VN) BAM format version.
fileData[].programs[] list (@PG) Programs.
fileData[].programs[].commandLine string (CL) Command line.
fileData[].programs[].id string (ID) Program record identifier.
fileData[].programs[].name string (PN) Program name.
fileData[].programs[].prevProgramId string (PP) Previous program id.
fileData[].programs[].version string (VN) Program version.
fileData[].readGroups[] list (@RG) Read group.
fileData[].readGroups[].date string (DT) Date the run was produced (ISO8601 date or date/time.)
fileData[].readGroups[].description string (DS) Description.
fileData[].readGroups[].flowOrder string (FO) Flow order. The array of nucleotide bases that correspond to the nucleotides used for each flow of each read.
fileData[].readGroups[].id string (ID) Read group identifier.
fileData[].readGroups[].keySequence string (KS) The array of nucleotide bases that correspond to the key sequence of each read.
fileData[].readGroups[].library string (LS) Library.
fileData[].readGroups[].platformUnit string (PU) Platform unit.
fileData[].readGroups[].predictedInsertSize integer (PI) Predicted median insert size.
fileData[].readGroups[].processingProgram string (PG) Programs used for processing the read group.
fileData[].readGroups[].sample string (SM) Sample.
fileData[].readGroups[].sequencingCenterName string (CN) Name of sequencing center producing the read.
fileData[].readGroups[].sequencingTechnology string (PL) Platform/technology used to produce the reads.
fileData[].refSequences[] list (@SQ) Reference sequence dictionary.
fileData[].refSequences[].assemblyId string (AS) Genome assembly identifier.
fileData[].refSequences[].length integer (LN) Reference sequence length.
fileData[].refSequences[].md5Checksum string (M5) MD5 checksum of the sequence in the uppercase, excluding spaces but including pads as *.
fileData[].refSequences[].name string (SN) Reference sequence name.
fileData[].refSequences[].species string (SP) Species.
fileData[].refSequences[].uri string (UR) URI of the sequence.
id string The readset ID.
name string The readset name.
readCount unsigned long The number of reads in this readset.

Methods

get
Gets a readset by ID.
search
Gets a list of readsets matching the criteria.
