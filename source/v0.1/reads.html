Reads

A Read is a group of bases that may or may not have been aligned to a reference. It contains quality information and other metadata. All Reads belong to exactly one Readset.

For a list of methods for this resource, see the end of this page.

Resource representations

{
  "id": string,
  "name": string,
  "readsetId": string,
  "flags": integer,
  "referenceSequenceName": string,
  "position": integer,
  "mappingQuality": integer,
  "cigar": string,
  "mateReferenceSequenceName": string,
  "matePosition": integer,
  "templateLength": integer,
  "originalBases": string,
  "alignedBases": string,
  "baseQuality": string,
  "tags": {
    (key): [
      string
    ]
  }
}
Property name Value Description Notes
alignedBases string The originalBases after the cigar field has been applied. Deletions are represented with '-' and insertions are omitted.
baseQuality string Represents the quality of each base in this read. Each character represents one base. To get the quality, take the ASCII value of the character and subtract 33. (QUAL)
cigar string A condensed representation of how this read matches up to the reference. (CIGAR)
flags integer Each bit of this number has a different meaning if enabled. See the full BAM spec for more details. (FLAG)
id string The read ID.
mappingQuality integer A score up to 255 that represents how likely this read's aligned position is correct. A higher value is better. (MAPQ)
matePosition integer The 1-based start position of the paired read. (PNEXT)
mateReferenceSequenceName string The name of the sequence that the paired read is aligned to. This is usually the same as referenceSequenceName. (RNEXT)
name string The name of the read. When imported from a BAM file, this is the query template name. (QNAME)
originalBases string The list of bases that this read represents (e.g. 'CATCGA'). (SEQ)
position integer The 1-based start position of the aligned read. If the first base starts at the very beginning of the reference sequence, then the position would be '1'. (POS)
readsetId string The ID of the readset this read belongs to.
referenceSequenceName string The name of the sequence that this read is aligned to. This would be 'X' for the X Chromosome or '20' for Chromosome 20. (RNAME)
tags object A map of additional read information. (TAG)
tags.(key)[] list A string which maps to an array of values.
templateLength integer Length of the original piece of dna that produced both this read and the paired read. (TLEN)

Methods

search
